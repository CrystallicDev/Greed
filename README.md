## Description
**Greed** is a mod that aim to rebalance some vanilla mechanics, without adding any, and without removing any. 

It is born out of a common complaint about Vanilla Minecraft : "**_There is no real endgame_**". 

A lot of mods try to expand on the endgame by adding some even stronger weapons and items, but i personally think that the game has already a great variation of tools and items. The main issue isn't the endgame, but the speed at which it is attainable. 

For example : 
* Why would you craft Copper Armor, if Iron is nearly as common ?
* Why would you use gold armor, if it is so fragile and weak ?
* Why would you enchant anything below "iron" grade, if you need a diamond pickaxe to make an enchanting table ?
* Why would you bother adding biomes to the end, if you need to complete the game to visit it ?
* Why would you bother making potions if it is so easy to defeat monsters ?
* Why would you bother making beetroot or carrot farms if you can just farm bread / potatoes / beef easiliy ?

To answer those questions, and more, **Greed** changes the way the **Enchanting Table** behave, the way **Cauldrons** and **Brewing Stands** works, the way **food** and **crops** works, to make gathering food an actual activity, and even reworks the vanilla progression !

> **Greed** can be used on its own, but is intended to be used alongside Cristallite, Natsu's Mobs, Jewels and Rings, and Elixirs.
